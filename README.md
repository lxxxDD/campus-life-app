## 🚀 快速开始 (Quick Start)

### 环境准备

- **HBuilderX**: 3.8+ (推荐)
- **Node.js**: 16+

### 运行步骤

```bash
# 1. 📥 克隆项目
git clone https://github.com/lxxxDD/campus-life-app.git

# 2. 📦 安装依赖
npm install

# 3. 🏃 运行项目
# 方式一：命令行
npm run dev:h5

# 方式二：HBuilderX
# 打开项目 -> 运行 -> 运行到浏览器/模拟器
```
